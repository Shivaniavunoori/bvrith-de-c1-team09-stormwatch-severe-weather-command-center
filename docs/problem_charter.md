## 1. Problem Context

StormWatch: Severe Weather Intelligence Hub is a data engineering project that monitors and analyzes severe weather events such as heavy rainfall, cyclones, floods, thunderstorms, heatwaves, and strong winds. It collects weather data from multiple sources, cleans and processes the data, and generates reliable insights. The final dashboard helps meteorologists, disaster management authorities, and government agencies monitor weather conditions, identify potential risks, and make informed decisions.

---

## 2. Engineering Problem

The project converts raw weather data such as temperature, humidity, rainfall, wind speed, atmospheric pressure, and weather alerts into clean and trusted Bronze, Silver, Data Quality, and Gold layers using Databricks. The processed data is then visualized in Power BI dashboards to provide real-time weather insights, detect severe weather conditions, and support disaster preparedness and decision-making.

## 4. Scope Inclusions

The team will build:

- Raw weather data collection
- Bronze data ingestion
- Silver data cleaning and standardization
- Data quality validation
- Gold weather metrics
- Power BI weather dashboard
- Real-time weather streaming simulation
- GitHub documentation and weekly evidence

---

## 5. Scope Exclusions

The team will **not** build:

- A live weather forecasting system
- Mobile or web applications
- Real-time satellite integration
- Emergency notification services
- Copied internet projects
- Fake screenshots or unexplained AI-generated work

---

## 6. Success Criteria

By the end of 12 weeks, the project is successful if:

- The complete weather data pipeline can be explained from end to end.
- The team can demonstrate Bronze, Silver, Data Quality, Gold, Dashboard, and Streaming layers.
- All three students can explain the complete project architecture and workflow.
- GitHub contains weekly documentation, screenshots, commits, and final submission files.
- Power BI dashboards display meaningful weather insights using Gold datasets.
- Streaming simulation successfully processes live weather events.
